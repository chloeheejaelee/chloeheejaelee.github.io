What is HTML?

A perfect example of what HTML is would be that it is the wood foundation of a mansion. To be able to build a web page, HTML helps construct and manipulate code in order to finalize a product of what you want your customers to see. HTML is an acronym which stands for Hyper Text Markup Language which is used for creating online pages and apps. We will delve deeper into what is meant by Hypertext Markup Language.

  • Hyper Text:Also known as "Text within Text." When a text has been connected with a link, it is known as a hypertext. So when the user clicks on the text and it sends them to a different webpage

  • Markup Language:A markup language is a computer language that is used to apply layout and formatting conventions to a text document. Markup Language makes text more interactive and dynamic. It can turn text into images, tables, links, etc.

  • Web Page:A web page is a document which is commonly written in HTML and translated by a web browser. A web page can be identified by entering an URL. A Web page can be of the static or dynamic type.

Therefore, HTML is a markup language which results in creative web pages by using different styling, formats, and layouts.

Basic Example of HTML

  • !DOCTYPE html:declaration deines that this document is an HTML5 document

  • html:element is the root element of an HTML page

  • head:element contains meta information about the HTML page

  • title:element specifies a title for the HTML page (which is shown in the browser’s title bar or in the page’ tab)

  • body:element defines the document’s body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

  • h1:element defines a large heading

  • p:element defines a paragraph

  • a href=”linkgoeshere.com:This is a link

  • img src=”imagegoeshere.jpg” alt=”linkgoeshere.com” width=”104” height=”142”:This allows the user to embed a link in an iamge. The width and height customizes the image.
